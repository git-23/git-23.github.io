# Head First HTML and CSS

## SECTION 1

- `<html>`告诉浏览器文件的内容是 HTML
- `<head>`包含 Web 页面的有关信息，放置`<title>`、`<style>`元素
- `<body>`包括了页面主体

## SECTION 2

- `<a>`元素通过`href`属性指定链接的目标文件，属性的属性值需要有引号
- 相对路径中使用`../`上行一个文件夹
- 可以把网页、图片和其它资源分别放在不同的文件夹中维护

## SECTION 3

- 块元素显示时就好像前后各有一个换行，而内联元素在页面文本流中总在“行内”出现
- 有的元素是没有内容的元素，被称为 void 元素，如`<br>`元素、`<img>`元素

## SECTION 4

- URL 可以用来定位 Web 上的任意资源，分为协议名，网站名，资源的绝对路径三部分
- 点击一个相对链接时，浏览器会根据相对路径和所点击页面的路径创建一个绝对路径
- 浏览器向 Web 服务器请求一个目录的时候，服务器会尝试查找目录中的默认文件
- 服务器接收到一个末尾没有`/`的请求，这个目录确实存在，服务器会帮你加上斜杠
- 浏览器从你的计算机本地读取文件的时会使用 file 协议
- 通常 Web 上的所有东西都会传送到一个默认端口(80)
- 要链接到页面中的一个特定目标，在链接最后加一个`#`，再加上目标标识符。

## SECTION 5

首先，浏览器从服务器获取 HTML 文件。接下来，浏览器读取 HTML 文件，显示这个文件，发现其中有四个图像需要获取。所以它需要从 Web 服务器逐个得到这些图像。

Web 上最常用的图片格式有三种，JPEG、PNG 和 GIF。

JPEG 最适合连续色调图像，如照片。可以表示多达 1600 万种颜色。这是一种有损格式，在缩小文件大小时会丢掉图像的一些信息。不支持透明度和动画。

PNG 适合单色图像和线条构成的图像。取决需要的颜色可以使用三种 PNG 中的一种。这是一种无损格式，在压损文件大小时不会丢失信息，允许将颜色设置为透明。

GIF 类似 PNG ，但可以表示最多 256 种不同颜色的图像。也是一种无损格式，支持透明度和动画。

使用`<img>`元素的`alt`属性提供候选文本。可以使用`height`、`width`属性指定高度和宽度，提供图片的信息，但最好不要进行缩放，结构和表现分离。

调整照片至适合的大小，而不是下载完整的图片后进行缩小，这可以需要更多的时间。

为透明图片设置与背景色相符颜色的蒙版。

## SECTION 6

使用`<!DOCTYPE html>`指定文档类型为 HTML5 文件。

> HTML5 是活的标准，具有着向后兼容性，表示我们可以继续向 HTML 增加新的内容，浏览器会支持这个新内容，不过他们还会继续支持原来的内容。

可以使用 [W3C 验证工具][1]检查你的 HTML 文档。

[1]: http://validator.w3.org "W3C 验证 HTML 文档的工具"

`<meta charset="utf-8">`中`<meta>`标记告诉浏览器关于页面的一些信息，`charset`属性指定字符编码，`"utf-8"`是 Web 页面中常用的编码。

> 编写合法的 HTML 页面 
> - 一定要以`<DOCTYPE>`开头
> - `<html>`元素，不能没有它
> - 记住，要使用`<head>`和`<body>`编写更好的 HTML
> - 在`<head>`中指定正确的字符编码
> - 一定要在`<head>`元素中包含一个`<title>`元素
> - 嵌套某些元素的时候要当心
> - 检查必要的属性

## SECTION 7

CSS 包含一些简单语句，称为规则。每个规则为选择的一些 HTML 元素提供样式。典型的规则包括一个选择器，以及一个或多个属性的值。

要为 HTML 直接增加 CSS 样式，需要在`<head>`元素中增加开始和结束`<style>`标记，你的 CSS 规则放在其中。

要为多个元素编写一个规则，只需要在选择器之间加上逗号。

创建一个 CSS 文件，在 HTML 中使用`<link>`元素可以链接到外部样式表。`rel`指定了 HTML 文件与所链接文件的关系，这里使用值`stylesheet`, `href`属性中放置 CSS 文件的位置，可以是相对链接，也可以是一个完整的 URL。`type`属性在 HTML5 中不再需要，但它是可选的。

不是所有的样式都能继承，只有一部分样式可以继承，如`font-family`。

可以使用更特定的选择器覆盖从父选择器继承的属性。

CSS 中可以使用`/*   */`指定注释。

可以为元素增加`class`属性，然后通过类选择器为其添加特定的样式。

多个元素可以指定相同的类，一个元素可以加入多个类。

W3C 提供了 [CSS 验证工具][2]。

[2]: http://jiqsaw.w3.org/css-validator/ "W3C 提供的 CSS 验证工具"

## SECTION 8

1. 用`font-family`属性定制页面中使用的字体。
2. 用`font-size`属性控制字体大小。
3. 用`color`属性为文本设置颜色。
4. 用`font-weight`属性影响字体的粗细。
5. 用`text-decoration`属性为文本增加更多风格。
    - line-through 从文本中间穿过的横线
    - underline
    - overline
    - none
    - 可以同时指定多个值，用空格分隔。
6. 用`font-style`为字体增加风格
    - italic 斜体风格
    - oblique 倾斜文本

每个`font-family`包含一组有共同特征的字体。共有五个字体系列
- sans-serif 字体系列包括没有衬线的字体，与 serif 相比，通常认为 sans-serif 字体在计算机上更容易识读。
- serif 字体系列包括有衬线的字体。
- monospace 字体系列中的字体包含固定宽度的字符，这些字体主要用于显示软件代码示例。
- cursive 字体系列包括看似手写的字体，有时会在标题中看到这些字体。
- fantasy 字体系列包含含有某种风格的装饰性字体。

使用`font-family`属性可以指定多个字体，只需要输入这些字体名，并用逗号分隔。

通常，你指定的`font-family`包括一个候选字体列表，他们都来自同一个字体系列。

最后总是放一个通用的字体系列名。

实际上就是一个字体通用列表，第一个是你最希望使用的。

为页面增加 Web 字体
1. 找到一个字体
2. 确保有所需字体的所有格式
3. 把你的字体文件放在 Web 上
4. 在 CSS 中增加`@font-face`属性
    - `font-family`属性指定一个 font-family 名，以供以后使用。
    - `src`属性告诉浏览器在哪里可以得到这个字体。对于浏览器可识别到的每一个文件，我们要分别指定一个 src 值，用逗号分隔。
5. 在 CSS 中使用`font-family`名
6. 加载页面

可以认为`@font-face`是一个内置的 CSS 规则，而不是一个选择器规则。

还有其他的内置 CSS 规则，如`@import`和`@media`。

额外的字体会额外的增加用户加载的时间。

指定字体大小的方式
- px 可以按像素指定大小，用像素指定大小时，就是在告诉浏览器字母高度是多少像素。
- % 用一个百分比指定字体大小时，会相对另外一个字体指出这个字体有多大。`font-size`是从父元素继承而来的一个属性，指定百分数字体大小时，就是相对于父元素的字体大小。
- em 类似百分数，也是一个相对度量单位，指定一个比例因子，同样与父元素进行比较。
- 关键字 可以把一个字体大小指定为 xx-small, x-small, small, medium, large, x-large, xx-large。
    - 每个大小约比前一个大20%
    - 大多数浏览器中 small 字体大小大约为 12px。

指定字体大小
1. 选择一个关键字，作为 body 规则中的字体大小
2. 使用 em 或百分数，相对于 body 字体大小指定其他元素的字体大小。

标题的默认大小取决于浏览器，一般来讲，h1 200%, h2 150%, h3 120%, h4 100%, h5 90%, h6 60%，相较于默认体文本字体。

body 中使用 em 或 % ，将以默认字体为参照物， 默认字体通常是 16 px。

指定 Web 颜色
- 按名字
- 按红、绿、蓝值 `rgb(80%, 40%, 0%)`
    - 也可以指定为 0-255 之间的数值`rgb(205, 102, 0)`
- 十六进制码 以`#`开头，每组2位数字分别代表颜色的红、绿、蓝分量。

有下划线的元素常被以为是链接元素，可以用下边框作为装饰。

`background-image`为元素设置背景图像，`background-repead`设置背景图像是否重复，`background-poaition`设置背景图像的位置。

## SECTION 10

`<div>`元素允许你将页面划分为逻辑区和逻辑分组

`text-align`属性设置文本对齐。会对块元素中的所有内联内容对齐。这个属性会被继承。

后代选择器`div p {}`  
子代选择器`div>p {}`

**CSS 属性快捷方式**
`padding: 0px 10px 30px 20px;`  
`margin: 0px 20px 20px 20px`  
*按照上右下左的顺序*

`background: white url() no-reapeat`
`border: thin solid #007e7e`
*可以采用任意顺序*  
*可以任取几个元素*  

`font: font-style font-variant font-weight font-size/line-height font-family`
*font-size 和 font-family 是必选的*  
*font-size 前的属性不要求顺序*

`<span>`采用与`<div>`类似的方法建立内联内容的逻辑分组。

可以设置内联元素的宽度，不过在对它进行定位之前，你可能注意不到宽度改变的效果。另外还可以对这些元素增加外边距、内边距和边框。如果一个内联元素的四周都增加外边距，只能看到左右两侧增加空间。你也可以对内联元素的上边和下边增加内间距，不过这个内间距不会影响包围它的其他内联元素间距，所以内边距会与其他内联元素重叠。

**根据元素的状态指定样式**  
`a:link {}`应用于未访问的链接、`a:visited {}`应用于已访问的链接、`a:hover {}`悬停在一个链接上时会应用这个选择器。

`a:focus {}`浏览器将焦点放在链接上时，`a:active {}`用户第一次点击一个链接时。

浏览器会仔细检查所有的`<a>`元素，然后把它们增加到正确的伪类中。

# SECTION 11

**float 布局**
1. 使用`float`属性
    ```css
    div {
        float: right;
        width: 200px;
    }
    ```
2. 使用`margin`属性为浮动元素留出空间
3. 使用`clear`属性解决底栏被覆盖问题
    ```css
    #footer {
        clear: right;
    }
    ```
*内联元素会避开浮动元素*

**jello 布局**  
锁定内容区的宽度，内容区不进行扩展

**定位**
绝对定位的元素可以修改`z-index`属性分层放置

**表格显示**
1. 选择一个容器
    ```css
    div {
        display: table;
        border-spacing: 10px;/*边框间距*/ 
    }
    ```
2. 确定表格的行
    ```css
    div {
        display: table-row;
    }
    ```
3. 确定各行的列
    ```css
    div {
        display: table-cell;
    }
    ```
